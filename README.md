# 🌸 Whispers Garden

Whispers Garden is a quiet, browser-based space where you can slow down and create something small and meaningful. Paint your own flower, add a short message or thought, and place it into a shared garden that grows with every whisper.

It is meant for moments when you want to pause, reflect, or express something gently without needing a full conversation.

## Getting Started

No setup needed. Just open and start creating.

**Run locally**
1. Download or clone the project  
2. Open `index.html` in your browser  

**Or use it online**  
https://khushibhavsar.github.io/whispers-garden/

Paint your flower, add a whisper if you want, and plant it in the garden.
